## Overview

This exhibit explores the history of transistor miniaturization and how Moore's Law described the rapid growth of computing power throughout the semiconductor age.

Beginning with the invention of the transistor in 1947, visitors will follow the progression of semiconductor technology through integrated circuits, VLSI systems, and modern nanometer-scale processors. The exhibit demonstrates how shrinking transistors enabled faster, smaller, and more powerful computers while also examining the engineering challenges faced as transistor dimensions approached atomic scales.

Rather than focusing solely on Moore's Law, the exhibit uses Moore's Law as a guiding narrative to explain the evolution of transistor technology and its impact on modern computing.

# Interactive Exhibit Overview

The exhibit will feature multiple interactive components designed to help visitors understand both the historical progression and technical challenges of transistor miniaturization.

---

## Interactive Component 1: Semiconductor Evolution Timeline

### Description

An interactive timeline showcasing major milestones in semiconductor history.

Visitors can navigate through different decades and select important events to reveal detailed information.

### Timeline Entries

- 1947 — Point-Contact Transistor
- 1958 — Integrated Circuit
- 1971 — Intel 4004
- 1982 — 1.5 μm Manufacturing Process
- 1995 — 350 nm Manufacturing Process
- 2004 — 90 nm Manufacturing Process
- 2011 — 22 nm FinFET
- 2020 — 5 nm Process Nodes
- Present Day — 3 nm and Beyond

### User Interaction

When a milestone is selected:

- Historical information appears
- Images and diagrams are displayed
- Transistor count information is shown
- Semiconductor process size is highlighted
- Historical significance is explained

### Technologies

- React
- Astro
- CSS Animations

---

## Interactive Component 2: Moore's Law Growth Simulator

### Description

A dynamic visualization demonstrating how transistor counts increased over time according to Moore's Law.

### User Interaction

Visitors can move a timeline slider between 1971 and the present.

As the slider moves:

- Transistor count updates
- Process node size updates
- Computing power comparison updates
- Graph visualization changes in real time

### Technologies

- React
- Chart.js
- Astro

---

## Interactive Component 3: How Small Is a Transistor?

### Description

An interactive scale comparison that helps visitors visualize modern transistor sizes.

### User Interaction

Visitors can compare transistor sizes against:

- Human Hair
- Grain of Sand
- Red Blood Cell
- Virus
- DNA Strand
- Modern 3 nm Transistor

Users can zoom in and out to see relative size differences.

### Technologies

- React
- CSS Transformations
- SVG Graphics

---

## Interactive Component 4: The Limits of Moore's Law

### Description

An interactive engineering challenge simulator.

### User Interaction

Visitors attempt to continue shrinking transistor sizes while balancing:

- Performance
- Power Consumption
- Heat Production
- Manufacturing Cost

The simulator demonstrates the physical and economic barriers faced by semiconductor engineers.

### Technologies

- React
- State Management
- Dynamic Data Visualization
