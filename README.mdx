# The Race to the Atom: Moore's Law and How Transistors Shrunk Over the Decades

## Topic Theme
Historical Computing

This exhibit explores the history of transistor miniaturization, starting from the 1950s to the present day, highlighting how Moore’s Law guided the semiconductor industry for decades.

Visitors will discover how transistors evolved from large-scale electronic components into nanometer-scale structures, enabling the development of modern processors, smartphones, artificial intelligence systems, and supercomputers.

---

## What This Exhibit Covers

- What is a transistor?
- What is Moore’s Law?
- Why did transistor sizes continue shrinking to modern nanometer scales?
- Is Moore’s Law still relevant today, given physical and economic limitations?
- What physical limits did engineers encounter when scaling transistors down to near-atomic sizes?

---

## Interactive Exhibit Overview

This exhibit features an interactive timeline showcasing the evolution of transistor technology throughout computing history. Visitors can explore key milestones and access detailed information about transistors introduced during each era.

Each timeframe includes an interactive 3D model of a transistor. Users can rotate the model along the X, Y, and Z axes using mouse or touch controls. These models are rendered in 3D, converted into JSX components, and displayed using React, allowing full interactive inspection from any angle.
