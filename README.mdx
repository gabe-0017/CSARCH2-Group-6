# The Race to Remember: How Computer Storage Evolved from Punch Cards to the Cloud

This exhibit explores the evolution of computer storage technologies, starting with punch cards and magnetic tape systems and progressing to modern day solid-state drives and cloud storage platforms.

Discover how the increasing demand for data storage drove innovations in capacity, speed, portability, and reliability. The exhibit highlights how storage devices evolved from room-sized systems capable of storing only a few kilobytes to modern technologies that store terabytes of information in devices that fit in a pocket.

### Exhibit Roadmap

- **General introduction** to computer storage and why it is important.
- **The history of computer storage**
  - How early computers stored information.
  - How storage capacity increased over time.
  - The technologies that replaced previous storage mediums, and why.
  - How modern storage devices such as SSDs work.
  - Cloud storage, and where data actually stored.
- **Future challenges** that upcoming storage technologies will face.

## Interactive Elements

### 1. Interactive Storage Timeline
An interactive timeline that presents major milestones in the history of computer storage. Explore the progression from punch cards and magnetic tape systems to floppy disks, hard drives, solid-state drives, and modern-day cloud storage.

Selecting a storage technology reveals:
- Historical background
- Release year
- Storage capacity
- Physical dimensions
- Advantages and limitations
- Real-world applications

*This will provide historical context and establish the progression of storage technologies over time.*

### 2. Interactive 3D Storage Device Models
An interactive 3D exhibit that lets you examine the internal components of both historical and modern storage devices. 

**Supported Devices:**
- Punch Card
- Magnetic Tape Reel
- Floppy Disk
- Hard Disk Drive
- Compact Disc
- Solid State Drive

*It helps visualize the physical mechanisms used to store information across different generations of technology.*

### 3. Physical Size vs Capacity Visualization
A game where users must store a given amount of data using technologies available in a selected decade.

Example: Store **1 GB** of data into:
- Punch Cards
- Magnetic Tape
- Floppy Disks
- CDs
- HDDs

*The system visually displays how much physical space would be required. It will also demonstrate the relationship between storage capacity and physical size throughout computing history.*
